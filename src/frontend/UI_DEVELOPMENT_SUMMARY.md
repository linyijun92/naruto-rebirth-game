### ✅ 任务 2: 主菜单实现

- 5 个菜单按钮（继续游戏、新游戏、加载存档、设置、关于）

5. **关于** → 显示关于信息（版本、描述）

**视觉设计**:

- 副标题："火之意志，永不熄灭"
- 按钮使用橙色品牌渐变

## 技术实现细节

### 动画系统
- 使用 `AnimationController` 控制动画
- 使用 `Tween` 定义动画范围
- 使用 `CurvedAnimation` 应用缓动曲线
- 支持 `FadeTransition`、`ScaleTransition`、`SlideTransition`

### 路由系统
- 使用 `MaterialApp` 的路由配置
- 支持命名路由
- 404 处理（跳转到主菜单）

### 状态管理
- 准备好与 Provider 集成
- 预留状态管理接口
