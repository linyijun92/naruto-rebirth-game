# 火影忍者重生 - 界面设计方案

## 1. 界面架构总览

### 1.1 界面层次结构
```
┌─────────────────────────────────┐
│          启动画面               │
├─────────────────────────────────┤
│          主菜单界面             │
├─────────────────────────────────┤
│  ┌─────────────────────────┐   │
│  │      游戏主界面         │   │
│  ├─────────────────────────┤   │
│  │  ├─ 剧情界面            │   │
│  │  ├─ 属性界面            │   │
│  │  ├─ 任务界面            │   │
│  │  ├─ 商店界面            │   │
│  │  ├─ 背包界面            │   │
│  │  ├─ 存档界面            │   │
│  │  └─ 设置界面            │   │
│  └─────────────────────────┘   │
├─────────────────────────────────┤
│          设置界面             │
└─────────────────────────────────┘
```

### 1.2 导航流程
```
启动画面
    ↓
主菜单（新游戏 / 继续游戏 / 设置）
    ↓
游戏主界面
    ├─ 点击剧情区域 → 剧情界面
    ├─ 点击属性按钮 → 属性界面
    ├─ 点击任务按钮 → 任务界面
    ├─ 点击商店按钮 → 商店界面
    ├─ 点击背包按钮 → 背包界面
    ├─ 点击存档按钮 → 存档界面
    └─ 点击设置按钮 → 设置界面
```

## 2. 启动画面（Splash Screen）

### 2.1 设计目标
- 展示游戏品牌
- 营造忍者氛围
- 快速加载资源

### 2.2 界面布局
```
┌─────────────────────────────┐
│                             │
│                             │
│         游戏 Logo           │
│      （火影忍者元素）        │
│                             │
│                             │
│      [加载进度条]           │
│      ▰▰▰▰▰▱▱▱ 62%         │
│                             │
│      加载中...              │
│                             │
│                             │
└─────────────────────────────┘
```

### 2.3 视觉元素
- **背景**：深灰色渐变 (#1C1C1C → #282828)，添加忍者符文装饰
- **Logo**：中央位置，带有火焰动画效果
- **进度条**：橙色渐变 (#FF6B00 → #FF8C00)，圆角 4px
- **加载文字**：灰色 (#BBBBBB)，14pt

### 2.4 动画效果
- Logo 淡入：500ms，ease-out
- 火焰效果：循环播放
- 进度条动画：平滑填充，300ms 缓动

- **背景**：深灰卡片 (#303030)
- **圆角**：12px
- **内边距**：16px
- **阴影**：0 2px 8px rgba(0, 0, 0, 0.2)
- **头像**：圆形，64×64px
- **属性条**：橙色渐变，6px 高度

#### 剧情预览区域
- **背景**：透明，带边框
- **边框**：1px 灰色 (#3C3C3C)

- **内边距**：16px

- **背景**：深灰色 (#282828)
- **图标按钮**：48×48px
- **图标颜色**：灰色（未选中）→ 橙色（选中）

### 4.4 交互设计
- **角色卡片点击**：跳转属性界面
- **剧情预览点击**：跳转剧情界面
- **功能图标点击**：跳转对应界面
- **长按图标**：显示快捷操作菜单

```
└─────────────────────────────────────────────┘
```

### 5.3 视觉元素

#### 场景背景
- **尺寸**：全屏显示
- **风格**：日式动漫风格
- **过渡**：场景切换时淡入淡出

#### 角色立绘
- **位置**：屏幕左/右侧
- **尺寸**：自适应，不超过屏幕高度 60%
- **动画**：说话时轻微摇晃
- **表情**：根据剧情动态变化

#### 对话框
- **背景**：深灰色 (#1C1C1C)，90% 不透明
- **圆角**：12px
- **内边距**：20px
- **边框**：1px 橙色 (#FF6B00)，发光效果
- **角色名**：橙色，加粗，14pt
- **对话文本**：白色，16pt Regular，行高 1.6
- **动作描述**：灰色 (#BBBBBB)，斜体，14pt

#### 选项按钮
- **尺寸**：高度 48px，宽度自适应
- **背景**：橙色渐变 (#FF6B00 → #FF8C00)
- **文字**：白色，16pt Medium
- **圆角**：8px
- **间距**：8px
- **选中效果**：边框发光

### 5.4 交互设计
- **点击对话框**：自动播放下一句
- **长按对话框**：暂停/继续自动播放
- **选项选择**：触发剧情分支
- **点击角色**：显示角色信息
- **跳过按钮**：快速跳过对话

### 5.5 动画效果
- **场景过渡**：淡入淡出，1000ms
- **角色入场**：从下向上滑入 + 淡入，500ms
- **文字逐字显示**：50ms/字
- **选项出现**：从下向上依次弹出，300ms 间隔
- **选择确认**：缩放 + 淡出，200ms
