# 火影忍者重生 - 配色方案

## 1. 色彩体系概述

本配色方案基于火影忍者原著的视觉风格，结合日式动漫美学，打造神秘、热血的忍者世界氛围。

### 1.1 色彩理论基础
- **主色调**：深灰色、橙色、红色
- **辅助色调**：蓝色、紫色、绿色
- **功能色**：金色（特殊标识）、白色（背景）、黑色（文字）

### 1.2 设计原则
- **对比度**：确保文字和背景之间有足够的对比度
- **层次感**：通过色彩明度建立视觉层次
- **情感传达**：色彩与游戏氛围、情节情绪相匹配
- **品牌识别**：橙色作为品牌主色，贯穿全游戏

## 8. 使用指南

### 8.1 色彩搭配原则

#### 单色搭配（Monochromatic）
- 同一色相，不同明度
- 适合：属性界面、任务界面
- 示例：橙色 #FF6B00 + #FF8C00 + #E66000

#### 互补色搭配（Complementary）
- 色相环相对色
- 适合：强调元素、对比区域
- 示例：橙色 #FF6B00 + 蓝色 #2196F3

#### 三分色搭配（Triadic）
- 色相环等距三色
- 适合：多彩界面、活动页面
- 示例：橙色 #FF6B00 + 绿色 #4CAF50 + 紫色 #9C27B0

### 8.2 各场景色彩推荐

#### 主界面
- 背景：深灰色 #1C1C1C
- 激活元素：橙色 #FF6B00
- 次要元素：灰色 #666666

#### 剧情界面
- 背景：深色渐变
- 对话框：白色 #FFFFFF 或深灰色 #303030
- 强调文字：橙色 #FF6B00 或红色 #E53935

#### 属性界面
- 背景：深灰色 #1C1C1C
- 属性条：各属性渐变色
- 数值：白色 #FFFFFF

#### 任务界面
- 主线：红色 #E53935
- 支线：橙色 #FF6B00
- 日常：绿色 #43A047

#### 商店界面
- 普通：灰色 #9E9E9E
- 稀有：蓝色 #2196F3
- 史诗：紫色 #9C27B0
- 传说：金色 #FFC107

## 10. 色彩对比度检查

### 10.1 符合 WCAG AA 标准的组合
以下组合对比度 ≥ 4.5:1：

- 白色文字 (#FFFFFF) + 黑色背景 (#1C1C1C) = ✅
- 白色文字 (#FFFFFF) + 橙色按钮 (#FF6B00) = ✅
- 灰色文字 (#666666) + 白色背景 (#FFFFFF) = ✅
- 橙色文字 (#FF6B00) + 白色背景 (#FFFFFF) = ✅

### 10.2 需注意的组合
以下组合对比度 < 4.5:1，需调整使用场景：

- 灰色文字 (#999999) + 深灰背景 (#303030) = ⚠️
- 浅色文字 (#E0E0E0) + 浅灰背景 (#F5F5F5) = ⚠️
